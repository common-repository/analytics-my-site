== Description ==

This analytics plugin allows you to enable Google Analytics on all of your pages almost instantly!

Simply enter in your analytics site ID and then this plugin will be active - Google Analytics should now track statistics on your Wordpress installation.

== Installation ==

1. Upload `analytics-my-site.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the Google Analytics Settings in your Administration Panel
4. Analytics stats will now be collected from your site.
